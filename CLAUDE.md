# streamdeck — CLAUDE.md

## What This Is
Python-based Stream Deck profile generator and installer for macOS. Creates .streamDeckProfile packages from Python definitions.

## Status
Stable utility. Used for FCP, OBS, and general automation profiles.

## Conventions
- Profile definitions in `profiles/` as Python scripts
- Installer: `~/streamdeck_installer.py`
- Device: Stream Deck 31 (3x5 = 15 keys)
- See README.md for full button types and plugin support
- OBS overlay profiles generated by `profiles/obs_overlays.py`
